#### 실행방법
1. news-feed/src/main/resources/application.properties에 본인이 사용할 DB를 연결한다.
2. gradle을 통해 import > build > bootRun 과정을 거쳐 실행한다.
3. baseUrl의 HomePass(예:http://localhost:5000/) 로 접근하면, build 시 테스트로 생성된 API문서를 확인할 수 있다.
